
1. What is FTP and Why Does It Exist? (Purpose in Real Life)
FTP = File Transfer Protocol
Its only job is: Send and receive files between two computers over the network.
Real-life uses (where you see it every day):

Old websites: Many small companies and old hosting (like some  web developers) still use FTP to upload website files (index.html, images, etc.) from their laptop to the web server.
Internal company file sharing: Banks, factories, government offices, and small businesses use FTP to move big files (reports, invoices, software updates) between departments.
Software companies: Some vendors (old ones) still send updates via FTP.
Backup systems: Some automatic backup tools upload files to a server using FTP.

Important fact: FTP is from 1985 — very old. It was made before security was a big thing, so everything travels in plain text (username, password, file content). That’s why it is dangerous in 2026.
